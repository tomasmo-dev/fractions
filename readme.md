---
`input` a `output` složky se automaticky vytvoří, pokud neexistují při spuštění programu.

---
Do `input` složky lze vložit libovolný počet jak `.csv` tak `.zip` souborů. Ty se následně zpracují a výsledky se uloží do `output` složky. \
`.zip` soubory ponechají původní strukturu, ale všechny `.csv` soubory se zpracují a uloží do `output` složky.
